We are developing a project named "Sitting Plan". Tools being used are C++, MySql database, make file, CGI to get the information from the browser. Purpose: Its main purpose to minimize the cheating in the exams. There are 3 strategies(Methods by which students of different groups are sitting). Source Code: The code of our project is at: https://github.com/satvirkaur/Sitting-Plan

Download the code and run it by using following procedure: 1.) Open the terminal 2.) Enter into the project directory 3.) run command make The provide the appropriate information and get sitting plan for your examination. Enjoy it :)
